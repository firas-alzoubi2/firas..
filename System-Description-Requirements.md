### **System Name: Smart Transport Trip Management System**

**System Description:**
A comprehensive software solution designed to streamline the management of smart transport operations. The system facilitates interaction between three main roles: Administrators, Drivers, and Passengers. It handles user accounts, vehicle fleet management, driver assignments, and the full lifecycle of trip scheduling, booking, and monitoring, along with a feedback system to ensure service quality.

**Top 14 Functional Requirements:**

1.  **User Registration:** Allow new users (passengers) to create a new account to access the system. *DONE*
2.  **Role-Based Login:** Secure authentication for Administrators, Drivers, and Passengers with appropriate access levels. *DONE*
3.  **User Profile Management:** Enable users to update their personal information and change their passwords securely. *DONE*
4.  **Admin User Management:** Allow administrators to add, search for, and edit user details within the system. *DONE*
5.  **Account Status Control:** Empower administrators to deactivate (suspend) or delete accounts when necessary. *DONE*
6.  **Driver Data Management:** Manage driver profiles, ensuring their records and license details are up to date. *DONE*
7.  **Vehicle Data Management:** Capabilities to add new vehicles to the fleet and edit existing vehicle information. *DONE*
8.  **Driver-Vehicle Assignment:** Functionality to link a specific driver to a specific vehicle for trip operations. *DONE*
9.  **Create New Trips:** Allow the scheduling and creation of new transport trips with defined routes and times. *DONE*
10. **Trip Administration:** Grant administrators the authority to edit trip details or cancel trips if required. *DONE*
11. **Trip Search:** Allow users to search for available trips based on specific criteria. *DONE*
12. **Trip Schedule Viewing:** Display a clear schedule of upcoming and completed trips for tracking purposes. *DONE*
13. **Rating System:** Enable passengers to rate both the driver and the vehicle after a trip to maintain quality standards.
14. **Secure Logout:** A function to safely terminate the user session for all account types. *DONE*
